# 🤖 Trash Cleaner Game

A fast-paced, 2D top-down roguelike game built with **Godot 4**. Take control of a cleaning robot to navigate an expanding map, collect dynamically spawning trash, and survive as long as possible before the garbage overwhelms you!

## ✨ Key Features

* **Dynamic Spawning System:** Trash items spawn randomly across the map. The game challenges the player to manage the escalating mess.
* **Player Progression:** Collecting trash grants XP, allowing for future upgrades and scaling difficulty.

* **Engine:** Godot Engine 4.x
* **Language:** GDScript
* **Art Style:** 2D Pixel Art

## 🎮 How to Play

* **Movement:** Use `W` `A` `S` `D` or `Arrow Keys` to move the robot.
* **Objective:** Pick up trash to earn XP and prevent the trash count from reaching the maximum limit.
* **Failure Condition:** If the amount of uncollected trash reaches the limit, the game ends instantly!
